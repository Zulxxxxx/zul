# zul
All-in-One Toolkit berbasis terminal (CLI) yang dirancang khusus untuk pengguna Termux di Android. Script ini menawarkan solusi praktis untuk manajemen file cloud dan pengunduhan media sosial dalam satu antarmuka yang profesional dan aman.

## Langkah-langkah:

1.  **Berikan Izin Eksekusi**
    Ketik perintah berikut agar script bisa dijalankan:
    ```bash
    chmod +x zul.sh
    ```

2.  **Jalankan Script**
    Gunakan perintah ini untuk memulai:
    ```bash
    ./zul.sh
    ```

---
*Catatan: Jika muncul error "Permission Denied", pastikan file berada di direktori home Termux, bukan di folder download internal HP.*
